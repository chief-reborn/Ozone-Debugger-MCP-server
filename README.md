# Ozone-Debugger-MCP-server

OK, It's just a MCP for those embedded software engineer who don't want to use the ugly Ozone GUI.
why don't we just let the agent do it

## feature

-  **Auto modify.jdebug file** 
-  **Designed for AI** - support the AI MCP
-  **log capture** - capture the Ozone output and log

## What do you need for this MCP

### software runtime
- **Python 3.7+**
- **SEGGER Ozone** 
- **J-Link driver** 

### hardware runtime
- **Jlink of course**
